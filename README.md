# MinnowSnap

MinnowSnap is a desktop screen capture tool built with Rust and Qt. It focuses on fast capture, lightweight editing, and local post-processing for screenshots.

![MinnowSnap screenshot](.github/assets/screenshot.png)

## Features

- Fast screen capture
- Simple annotation
- Local OCR
- QR code detection
- Copy, save, and pin
